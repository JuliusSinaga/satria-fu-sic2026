# Satria DATA - SIC 2026

## Deskripsi Proyek

Proyek ini disusun untuk mengikuti Statistics Infographic Competition (SIC) 2026. Tim mengangkat topik analisis transisi kendaraan konvensional ke kendaraan listrik dari perspektif finansial personal melalui pendekatan Total Cost of Ownership (TCO), Break Even Point (BEP), dan Future Value (FV).

Tujuan utama proyek adalah memberikan gambaran berbasis data mengenai dampak ekonomi penggunaan kendaraan listrik dibandingkan kendaraan berbahan bakar bensin dalam jangka pendek maupun jangka panjang.
